Types of Values in JavaScript

There are mainly six types of values in JS that are - Numbers,Strings,Boolean Values,Null,Un-defined Each values have diffrent meaning and also diffrent purpose to solve.

Values

  1. Numbers-Very simply in Layman terms these values are simple numeric values which we use in ou day to day life . These values are mainly use for arithmetic operations ( multiplication , addition ,subtraction , division and so on).There are also three special numbers that are Infinity , -Infinity , NAN (not a number)
  2. Strings text-Its simple as words written inside quotes("") are the Strings. You can have write Strings inside double-quotes(""),single-quotes(''),back-ticks(``).VArious string properties consists of Escaping which basically helps to change the line without escaping from the quotes or back-ticks that is achieved by using backslash.Strings can't be multiplied or divided any text can't be from very definition they just can be concatenated(that is joined) by using "+" symbol.
  3. Boolean Values (True/False)This kind of values are used for comparision where we compare values.ComparisonThese values are comparing in between similar kind of values
  4. Empty Values-Null and Un-definedThese are two special values that are used to denote the absence meaningful value.They are values but they carry no information.

Automatic Type Conversion

Sometimes when operators are applied on wrong types of values JS will convert those to required types by following some set of rules this is called Type Coercionlike concatinating a number written in string format with a Number value it concatenates but results vary once you subtract a number from a number written in a string value thats where type coercion comes in the picture

Unary Operator

All operators are not symbols.One of the example of this is typeOf which returns the type of the value as the name suggests. Operators that use two values are called Binary Operators, while those that take one are called Unary Operators

Logical-Operators

Logical Operators are the operations that can be applied to Boolean-Values themselves.Javascript support three logical operators and,or and not

  1. &&-This represents logical and.It is binary operator and its result is true if both the values are true otherwise false
  2. ||-This is a logical or. If either values are true it returns true.
  3. !-This is value which produces these kind or results !true = false and !false = true.That are in the way saying notTrue and notFalse.
